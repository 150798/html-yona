How To Order?


1. Buka Web pecetiksemarang.com atau install aplikasi Pecetik Semarang

2. Pilih Menu > Klik input jenis Handphone sesuai dengan Merk Handphone anda

3. Jangan lupa catat kode pemesananya ya:)

4. Masuk pada pilihan Order > Input data anda ke menu Order

5. Jangan lupa upload bukti transfer ya...

6. Thankyou!Pesanan kamu segera kami packing:)
